== Description ==

Kebbet Global Trash adds a trash page to the tools menu, that handles all trashed items on a site.

= Features =

* Wordpress plugin for handling all trashed items on one page.
* Filter each post type.
* Row actions per item (Delete permanently and Reset).
* Bulk-action to remove.
* Quick buttons for removing all, or all in a post type.
* One UI-page.

== Installation ==

Upload the Kebbet Global Trash plugin to your site, activate it, and then go the the tools page to handle all trashed items.
